# 日本語言語パック / Japanese Language Pack

逃离后室存档工具（Escape the Backrooms Save Tool）の日本語翻訳プラグインです。

## インストール方法

1. アプリの設定 → プラグインマーケットを開く
2. 「ローカルプラグインをインストール」をクリック
3. このフォルダを選択するか、`plugin.json` ファイルをドラッグ＆ドロップ

## ファイル構成

```
lang-ja-JP/
├── plugin.json        # プラグインメタデータ
├── translations.json  # 翻訳データ
└── README.md          # このファイル
```

## 翻訳への貢献

翻訳の改善や追加にご協力いただける場合は、`translations.json` を編集してプルリクエストを送信してください。
